# ai-btc-predictor

Project documentation lives in the [docs/](docs/) directory. Start with [docs/README.md](docs/README.md) for an overview, project structure, usage instructions, testing guidance, and data contract details.

## Quickstart

1. Install dependencies:
   ```bash
   poetry install
   ```
2. Explore the documentation:
   ```bash
   open docs/README.md
   ```
3. Run the CLI (example):
   ```bash
   poetry run alpha --help
   ```

Refer to the documentation folder for detailed guides on ingestion, feature engineering, training, inference, and monitoring workflows.
