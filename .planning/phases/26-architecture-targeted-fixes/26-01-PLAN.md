1. Rename `_generate_price_features` to `generate_price_features` (remove underscore prefix).
2. Rename `_join_auxiliary_features` to `join_auxiliary_features` (remove underscore prefix).
3. Add backward-compat aliases at module level: `_generate_price_features = generate_price_features` and `_join_auxiliary_features = join_auxiliary_features`. This prevents breakage if any untracked callers still use the old names.
4. Update the internal call sites within `build_xy` to use the new public names.

1. Add re-exports: `from bitbat.dataset.build import generate_price_features, join_auxiliary_features, build_xy`.
2. Add `__all__` listing these public names.

Create `src/bitbat/io/prices.py`:
1. Create a `load_prices(data_dir: Path, freq: str) -> pd.DataFrame` function that unifies the 3 divergent price loading implementations. It must:
   - Accept a `data_dir` (Path) and `freq` (str).
   - Scan `data_dir / "raw" / "prices"` for all `**/*.parquet` files (date-partitioned and flat legacy).
   - Filter frames that contain both `timestamp_utc` and `close` columns.
   - Concat, deduplicate on `timestamp_utc` (keep last), normalize timestamps to tz-naive UTC, set index to `timestamp_utc`, sort.
   - Raise `RuntimeError` if no price data found.
2. Also create a `load_prices_for_cli(freq: str) -> pd.DataFrame` convenience wrapper that uses the CLI's `_data_path` / default config-based data dir approach: loads from `data/raw/prices/btcusd_yf_{freq}.parquet` specifically, runs `ensure_utc`, sets index, sorts. This preserves the CLI's simpler single-file semantics.

In `src/bitbat/io/__init__.py`:
1. If not already re-exporting, add: `from bitbat.io.prices import load_prices`.

Note: The CLI (`_load_prices_indexed`) loads a single specific file (`btcusd_yf_{freq}.parquet`) while predictor/continuous_trainer scan all parquet files. The shared `load_prices` function handles both patterns via its glob approach (it will find the legacy flat file too). But `load_prices_for_cli` is a thin wrapper preserving the CLI's explicit single-file contract for callers that need it.
