<objective>
Empirically assess OBV fold-boundary leakage (LEAK-01) and, if confirmed material, implement a fold-aware OBV cumsum fix (LEAK-02).

Purpose: The OBV indicator uses `cumsum()` over the entire dataset, which means bars in later folds incorporate volume-direction information from all prior data including bars that would be in the "future" relative to a given fold's training window. If this leakage is material (measurably inflates model performance), the cumsum must be reset at fold boundaries.

Output: Assessment test with empirical comparison, conditional fix to OBV computation
</objective>

**Add fold-aware OBV variant to `src/bitbat/features/price.py`:**

Keep the existing `obv()` function unchanged (it's the standard OBV, correct for single-series use). Add a new function:

```python
def obv_fold_aware(
    close: pd.Series,
    volume: pd.Series,
    fold_boundaries: list[int] | None = None,
) -> pd.Series:
    """On-Balance Volume with cumsum reset at fold boundaries.

    When fold_boundaries is None or empty, behaves identically to obv().
    When fold_boundaries is provided, the cumulative sum resets at each
    boundary index, preventing information leakage across folds.
    """
    close_series = pd.Series(close, dtype="float64")
    volume_series = pd.Series(volume, dtype="float64").fillna(0.0)

    price_change = close_series.diff().fillna(0.0)
    direction = price_change.apply(lambda x: 1 if x > 0 else (-1 if x < 0 else 0))
    obv_contrib = volume_series * direction

    if not fold_boundaries:
        result = obv_contrib.cumsum()
    else:
        # Reset cumsum at each fold boundary
        segments = []
        sorted_boundaries = sorted(set(fold_boundaries))
        # Add 0 at start and len at end if not present
        if sorted_boundaries[0] != 0:
            sorted_boundaries = [0] + sorted_boundaries
        if sorted_boundaries[-1] != len(close_series):
            sorted_boundaries.append(len(close_series))

        for i in range(len(sorted_boundaries) - 1):
            start = sorted_boundaries[i]
            end = sorted_boundaries[i + 1]
            segment = obv_contrib.iloc[start:end].cumsum()
            segments.append(segment)

        result = pd.concat(segments)

    result.name = "obv"
    return result
```
