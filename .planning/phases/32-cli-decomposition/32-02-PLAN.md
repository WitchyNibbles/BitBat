For each of the 8 groups, create a new file in `src/bitbat/cli/commands/`. Copy the group declaration and all its commands verbatim from `cli/__init__.py`. Replace any reference to the old global group variable with the new local group variable. Import only what the commands in that module actually use from `bitbat.cli._helpers` and domain modules.

Key rules per module:
- `from bitbat.cli._helpers import ...` — only helpers actually called in this module
- Do NOT import from `bitbat.cli` or `bitbat.cli.__init__` — that creates circular imports
- Domain function imports (e.g., `from bitbat.ingest import prices as prices_module`) go directly in the command module, not via `_helpers`
- Keep function names identical (prices_pull, news_pull, features_build, etc.)
- Keep all Click decorators, options, help strings identical

Group line ranges to extract from `cli/__init__.py`:
- prices: group declaration + `prices.command("pull")` block
